# PPT制作1-快捷键

| 快捷键              | 作用                      |
| ------------------- | ------------------------- |
| CTRL+M              | 快速新建幻灯片            |
| SHIFT+鼠标拖拽      | 生成一个等比例（1:1）绘图 |
| CTRL+鼠标拖拽       | 中心放大                  |
| CTRL+SHIFT+鼠标拖拽 | 等比例中心放大            |
| CTRL+]              | 字体放大                  |
| CTRL+[              | 字体缩小                  |
