# Python模块5--string

1. ascii_letters

   + 功能

     生成全部字母，包括a-z,A-Z

   + 实例

     ```python
     import string
     print(string.ascii_letters)
     #abcdefghijklmnopqrstuvwxyzABCDEFGHIJKLMNOPQRSTUVWXYZ
     ```

2. digits

   + 功能

     生成数组，包括0-9

   + 实例

     ```python
     import string
     print(string.digits)
     #0123456789
     ```
