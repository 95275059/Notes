# Linux笔记59--系统管理3-工作管理

1. 把进程放入后台

   + 在命令后加&：进程在后台继续执行

     例：tar -zcf etc.tar.gz /etc &

   + 在进程执行过程中，按ctrl+z快捷键：进程在后台暂停

2. 查看后台工作

   jobs [-l]

   + -l	:	显示工作的PID

   注：“+”号代表最近一个放入后台的工作，也是工作恢复时，默认恢复的工作。

   注：“-”号代表倒数第二个放入后台的工作

   注：前面的数字是==工作号==

3. 将后台暂停的工作恢复到前台执行

   fg %==工作号==

   注：%可以省略，但注意是工作号，不是PID

4. 把后台暂停的工作恢复到后台执行

   bg %工作号

   注：后台恢复执行的命令，是不能和前台有交互的，否则不能恢复到后台执行
