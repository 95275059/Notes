# MATLAB笔记4--脚本文件

1. M文件

   MATLAB允许编写两种程序文件

   + 脚本

     脚本文件是以.m为扩展名的程序文件。

     在脚本文件中，可以编写一系列要一起执行的命令。

     **脚本文件不接受输入，不返回任何输出**

     **它对工作空间的数据进行操作**

   + 函数

     函数文件也是以.m为扩展名的程序文件

     函数可以接受输入和返回输出

     内部变量是函数的局部变量

2. 创建和运行脚本文件

   + edit

     创建一个Untitled的空文件

   + edit newfile.m

     创建一个名为newfile.m的新文件

   + mkdir 文件夹

     在当前目录中创建新文件夹
